# 📱 Kotlin App Development

Este repositório foi criado com o objetivo de documentar minha jornada no aprendizado de desenvolvimento de aplicativos utilizando **Kotlin**.

## 🚀 Sobre o projeto

Este projeto faz parte dos meus estudos em desenvolvimento mobile, com foco na linguagem Kotlin. Aqui você encontrará exemplos práticos, exercícios e pequenos projetos desenvolvidos ao longo do curso.

## 🧠 Objetivos

* Aprender os fundamentos da linguagem Kotlin
* Desenvolver aplicativos Android
* Entender conceitos como:

  * Programação orientada a objetos (POO)
  * Manipulação de dados
  * Interfaces gráficas (UI)
  * Ciclo de vida de atividades
* Aplicar boas práticas de desenvolvimento

## 🛠️ Tecnologias e Ferramentas

* Kotlin
* Android Studio
* Git & GitHub
